# Thermal Analysis of Disc Brake

This project focuses on the thermal analysis of an automotive disc brake system. During braking, kinetic energy of the vehicle is converted into heat due to friction between the brake pads and disc rotor. This study investigates the temperature distribution and heat dissipation characteristics of the disc brake.

---

## Project Objective

- Understand heat generation during braking
- Study temperature distribution on the disc brake
- Analyze thermal stresses caused due to braking
- Improve knowledge of brake system thermal behavior

---

## Tools Used

- CAD Software (SolidWorks / Fusion 360 / Similar)
- Simulation Tool (ANSYS / Thermal Analysis Software)
- MATLAB / Excel (for calculations and graphs)

---

## Methodology

1. Create a CAD model of the disc brake rotor.
2. Define material properties of the brake disc.
3. Apply braking heat flux conditions.
4. Perform thermal analysis using simulation tools.
5. Analyze temperature distribution and heat dissipation.
